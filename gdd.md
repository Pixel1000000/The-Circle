# Game Design Document
## Рабочее название: *The Circle*
**Жанр:** Roguelite / Top-down action  
**Сеттинг:** Тёмное фэнтези  

---

## 1. Концепция мира

Мир игры — идеальный **круг**, разделённый на **4 биома**-четверти. В самом **центре** находится комната финального босса. Биомы окружены стенами и изолированы друг от друга — переход возможен только через систему ключей.

```
        [ Биом 2: Пустыня ]
[ Биом 1: Лес ]  [★]  [ Биом 3: Зима ]
        [ Биом 4: Мёртвые земли ]
★ = Комната финального босса (рандомная тема из биомов 1–4)
```

Каждый биом занимает **открытое пространство** своей четверти круга с процедурной генерацией окружения, врагов и лута.

---

## 2. Прогрессия по миру

### Система ключей
- Мобы каждого биома **дропают фрагменты ключа** с определённым шансом.
- Для открытия ворот в следующий биом нужно **5 фрагментов**.
- Финальный босс доступен **только** из Биома 4.

### Уровень сложности биомов
| Биом | Тема | Уровень экипировки |
|------|------|-------------------|
| 1    | Лес / Поляна | Tier 1       |
| 2    | Пустыня | Tier 2            |
| 3    | Зима | Tier 3             |
| 4    | Мёртвые земли | Tier 4      |
| Центр | Комната босса (рандомная тема из биомов 1–4, **4 уникальных босса**) | — |

---

## 3. Боевая система

**Смешанный бой:** ближний + дальний (зависит от экипировки).

### Инвентарь
- Экипировка дропается с мобов и имеет **уровень (Tier 1–4)**, соответствующий биому.
- Совокупные характеристики **полного сета Tier N** выравнивают игрока с мобами биома N.
- Это создаёт органичный баланс: можно идти вперёд недоукомплектованным, но рискованно.

### Характеристики экипировки

Каждый предмет имеет **базовые статы** зависящие от слота и Tier'а. Полный сет Tier N выравнивает игрока с мобами биома N.

#### Броня (Шлем / Кираса / Поножи)
| Стат | Шлем | Кираса | Поножи |
|------|------|--------|--------|
| Броня (снижение урона) | ✓ (малая) | ✓ (основная) | ✓ (малая) |
| Макс. HP бонус | ✓ | — | — |
| Скорость передвижения | — | — | ✓ |

#### Оружие
| Стат | Меч + Щит | Лук |
|------|-----------|-----|
| Урон | ✓ | ✓ |
| Скорость атаки | Средняя (фиксировано) | Медленная (фиксировано) |
| Блок (% снижение урона при блоке) | ✓ | — |
| Дальность атаки | Ближняя | Дальняя |

> Меч+щит — надёжный и прощающий стиль. Лук — высокий урон, но уязвим в ближнем бою.

#### Значения по Tier'ам (примерные)
| Tier | Броня суммарно | Урон оружия | Бонус HP |
|------|---------------|-------------|----------|
| 1 | 10–15 | 8–12 | +10 |
| 2 | 20–28 | 18–24 | +20 |
| 3 | 35–45 | 32–40 | +35 |
| 4 | 55–70 | 52–65 | +55 |

Рост нелинейный — каждый следующий Tier даёт заметно больше, отражая резкий скачок сложности биомов.

#### Уникальные бонусы (редкий дроп)
Часть предметов может иметь один случайный бонус-стат:
- +% скорость атаки
- Жизнекрадство (небольшой хил от урона)
- +% урон ядом / льдом / огнём (тематически по биому)
- +% скорость передвижения
| Слот | Описание |
|------|---------|
| Шлем | Защита, возможно бонусы к характеристикам |
| Кираса | Основной защитный слот |
| Поножи | Защита, возможно бонус к скорости |
| Оружие | **Меч + щит** (ближний бой, блок) или **Лук** (дальний бой) |

### Зелья
- Механика à la Path of Exile: зелье **восстанавливается зарядами от убийств**, не подбирается с земли.
- Один слот зелья — принуждает к активному бою, исключает "копить на босса".

### Зелья
- Базовая система: зелья здоровья, маны/энергии.
- Дропаются с мобов или находятся в сундуках.
- Ограниченный инвентарь — принуждает к тактическим решениям.

## 4. Meta-progression

После каждого рана игрок получает **баллы опыта** в зависимости от:
- Количества убитых врагов
- Пройденных биомов
- Собранных фрагментов ключа
- (опционально) найденного лута

Баллы тратятся на прокачку **базовых характеристик**:

| Характеристика | Эффект |
|----------------|--------|
| Сила           | Увеличивает урон |
| Выносливость   | Снижает получаемый урон / увеличивает стамину |
| Здоровье       | Увеличивает макс. HP |

---

## 5. Персонаж

- Имя задаётся игроком перед началом рана.
- Скиллы и активные способности **не предусмотрены** — прогрессия только через экипировку и meta-характеристики.

---

## 6. Механика смерти

- Смерть → **главное меню**.
- Новый ран начинается **полностью с нуля** (экипировка, фрагменты ключа).
- Сохраняются только **прокачанные meta-характеристики** (Сила, Выносливость, Здоровье).

## 8. Мобы по биомам

### Биом 1 — Лес / Поляна
| Моб | Тип | Описание |
|-----|-----|---------|
| Волк | Ближний | Быстрый, атакует рывком |
| Гоблин-лучник | Дальний | Стреляет с дистанции, убегает при сближении |
| Энт (древесный страж) | Ближний, танк | Медленный, много HP, бьёт по области |
| Рой ос | Дальний | Много маленьких юнитов, роятся вокруг игрока |

### Биом 2 — Пустыня
| Моб | Тип | Описание |
|-----|-----|---------|
| Скорпион | Ближний | Ядовитая атака, накладывает DoT |
| Песчаный дух | Дальний | Бросает сгустки песка, может ненадолго ослеплять |
| Мумия | Ближний, танк | Медленная, много HP, при смерти взрывается бинтами |
| Наездник на ящерице | Ближний + мобильный | Быстро перемещается, таранит игрока |

### Биом 3 — Зима
| Моб | Тип | Описание |
|-----|-----|---------|
| Ледяной гоблин | Ближний | Хрупкий, но атакует группами |
| Снежная ведьма | Дальний | Ледяные снаряды, может замедлять (как финальный босс) |
| Йети | Ближний, танк | Очень медленный, но огромный урон и область атаки |
| Ледяной дух | Дальний | Невидим пока не атакует, снаряды рикошетят от стен |

### Биом 4 — Мёртвые земли
| Моб | Тип | Описание |
|-----|-----|---------|
| Скелет-воин | Ближний | Базовый, возрождается один раз если не уничтожить кости |
| Скелет-лучник | Дальний | Стреляет отравленными стрелами |
| Призрак | Дальний, мобильный | Проходит сквозь препятствия, наносит урон прикосновением |
| Костяной голем | Ближний, танк | Огромный, при смерти разлетается на скелетов-воинов |
